Node-Geolocalisation
====================

## Installation (nodejs library)

    npm install node-geocoder

## Usage example

```
distanceBetweenTwoCoord('-1,-2','3,4');
```
```
adressToCoord('Paris, France','New-York city');
```
```
distanceBetweenTwoAdress('Paris, France','New-York city');
```
